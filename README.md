# EngageU

EngageU is a comprehensive web application designed to streamline the management of university events and club activities. Built with the MERN stack, it serves as a centralized platform for students to explore campus life, register for events, and apply for club memberships, while providing administrators with tools to manage these activities efficiently.

## üöÄ Features

- **User Authentication**: Secure Signup and Login functionality using JWT (JSON Web Tokens).
- **Event Management**:
  - Browse upcoming university events.
  - One-click event registration for students.
  - Detailed event views.
- **Club Management**:
  - Explore various student clubs and organizations.
  - Apply for club memberships directly through the portal.
- **Admin Dashboard**:
  - Dedicated admin panel to oversee platform activity.
  - View and manage all event registrations.
  - Review and process club membership applications.
- **Responsive Design**: Modern and responsive UI built with Tailwind CSS for a seamless experience across devices.

### Frontend
- **React.js**: Library for building user interfaces.
- **Vite**: Next Generation Frontend Tooling for fast build and development.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **React Router**: For seamless navigation and client-side routing.
- **Axios**: For making HTTP requests to the backend.

### Backend
- **Node.js**: JavaScript runtime environment.
- **Express.js**: Web application framework for Node.js.
- **MongoDB**: NoSQL database for storing application data.
- **Mongoose**: ODM (Object Data Modeling) library for MongoDB and Node.js.
- **JWT (JSON Web Token)**: For secure user authentication and authorization.
- **Bcrypt.js**: For password hashing and security.
